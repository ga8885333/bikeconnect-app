# 騎騎 (ChiChi) - 摩托車社交應用程式

一個專為摩托車愛好者設計的現代化社交平台，提供騎行記錄、社群互動、揪團活動和地圖導航等功能。

## 🏍️ 項目概述

**騎騎 (ChiChi)** 是一個基於 React 18 開發的摩托車社交應用程式，旨在為摩托車愛好者提供一個完整的數位化騎行體驗平台。應用程式採用現代化的技術棧和用戶友好的設計，支持社群互動、活動組織、路線導航等核心功能。

## 🎯 核心功能

### 主要頁面
- **首頁 (HomePage)**: 個人騎行統計、快速操作、週進度顯示
- **社群 (SocialPage)**: 動態分享、互動評論、用戶關注
- **揪團 (GroupsPage)**: 活動創建、參與管理、活動分類
- **地圖 (FreeMapPage)**: 免費地圖導航、路線規劃、地點標記
- **個人 (ProfilePage)**: 個人資料、騎行歷史、成就展示
- **登入 (AuthPage)**: 用戶註冊、登入驗證

### 特色功能
- 📊 **騎行數據追蹤**: 距離、時間、速度等統計
- 🗺️ **智能地圖導航**: 基於 OpenStreetMap 的免費導航
- 👥 **社群互動系統**: 動態發布、點讚評論、私訊功能
- 🎯 **活動組織平台**: 創建和參與摩托車騎行活動
- 🏆 **成就系統**: 騎行里程碑和成就徽章
- 📱 **響應式設計**: 完美適配移動設備

## 🚀 快速開始

### 安裝與運行
```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm start

# 訪問應用
# Local: http://localhost:3000
# Network: http://192.168.1.190:3000
```

### 構建部署
```bash
# 構建生產版本
npm run build

# 部署到靜態服務器
npm run deploy
```
